# Course Overview

This course focuses on practical software development using **Design Patterns** and object-oriented design principles.

Students learn how to analyze real-world software problems and apply well-known design patterns to build maintainable and scalable applications.

The course also introduces advanced features of **C# and .NET**, along with architectural concepts such as UML modeling and multi-threaded programming.

# Exercise 01 – Facebook Desktop Application

The first assignment focuses on developing a **desktop application that interacts with the Facebook Graph API**.

## Goals

- Practice working with an external API (Facebook API)
- Practice UML diagrams
- Apply object oriented design
- Develop a desktop application using C# and .NET

---

## Requirements

The application must:

- Provide a **basic Facebook user experience**
- Include **at least two additional features** that do not exist in Facebook itself
- Be implemented using proper **object oriented design**

---

## Required UML Diagrams

The project must include the following diagrams:

- Use Case Diagram
- Two Sequence Diagrams (for the two new features)
- Class Diagram describing the system classes and their relationships

---

## Submission Requirements

The submission must include:

- The application source code
- One **DOCX document** containing:
  - UML diagrams
  - explanation of the implemented features
  - screenshots of the application

The DOCX file must be located in the same folder as the `.sln` file and have the same name.
